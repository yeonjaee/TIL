## groupby

`그룹연산`

```python
kimchi.groupby(by=None,  # 그룹핑 할 컬럼(기준)
               axis= 0,  # 그룹핑 연산 방향
               level = None) # 멀티 인덱스일 경우, 특정 레벨의 값을 그룹핑 컬럼으로 사용
```

- 성별 성적 평균, 학년별 성적 최고점수, 부서별 평균 연봉 등등을 구하는데 사용할 수 있음.
